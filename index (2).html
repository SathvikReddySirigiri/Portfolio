Publications

IEEE Confluence 2024 · Cloud Computing, Data Science & Engineering
Augmenting Clinical Decisions with Deep Learning: Lung Cancer Image Abnormality Segmentation

Developed a hybrid VGG16 + SVM model for lung cancer classification across benign, malignant, and normal cases using the IQ-OTH/NCCD dataset of 1,190 CT scan images in Python and TensorFlow, achieving 89.36% accuracy and outperforming CNN, DNN, and MobileNet baselines. Built a complete data preprocessing and augmentation pipeline applying normalization, rotation, horizontal flipping, and resizing to 224×224 pixels using Keras and TensorFlow to improve model generalization and reduce overfitting on limited annotated medical data. Evaluated and compared seven model architectures including ResNetv2, InceptionResNet, MobileNet, and VGG16-SVM using Precision, Recall, and F1-Score metrics, demonstrating that the hybrid VGG16-SVM model achieved the highest recall of 91.78% for malignant case detection, supporting more accurate clinical decision making.

IEEE Conference Publication
Advancements in Pulmonary Nodule Analysis: Unveiling Patterns and Enhanced Visualization

Developed a hybrid UNet + CNN framework for pulmonary nodule detection and lung cancer diagnosis on the LIDC-IDRI dataset of 15,548 CT scan images using TensorFlow and Keras, implementing Dropout, BatchNormalization, and L2 regularization to prevent overfitting. Processed and partitioned DICOM format CT images into 80/10/10 train-validation-test splits using pydicom and skimage, applying data augmentation and stratified sampling to ensure balanced nodule and non-nodule class representation. Evaluated the proposed UNet-CNN model against four baseline architectures including CNNVisAttention, CNN-AT-LSTM, and standalone UNet, achieving the highest performance with 97.67% precision, 97.81% recall, and 98.14% accuracy. Applied explainability techniques including Grad-CAM, IntGrad, ScoreCAM, and SmoothGrad to visualize model attention maps and support clinical trust in AI-driven lung cancer diagnosis.
